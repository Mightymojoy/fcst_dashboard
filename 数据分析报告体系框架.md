## 一、报告体系架构设计

### 1.1 三层报告体系

```
┌─────────────────────────────────────────────────────────┐
│  战略层：高管驾驶舱 (Executive Dashboard)               │
│  - 周期：日报/周报/月报                                 │

└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│  执行层：运营明细报表 (Operational Reports)             │

```

## 二、战略层报告（高管驾驶舱）

### 2.1 核心指标卡（每日更新）

| 指标类别 | 关键指标 | 计算逻辑 | 预警阈值 |
|---------|---------|---------|----------|
| **销售业绩** | 日均销售额 | 当日GMV/当月天数 | < 目标值90% |
| | 月度目标达成率 | 实际GMV/月度目标 | < 85% |
| | 同比增长率 | (本期-同期)/同期 | < 0% |
| **渠道表现** | 各渠道销售占比 | 渠道GMV/总GMV | 波动>5% |
| | 渠道ROI | 渠道GMV/渠道投入 | < 2.5 |
| **产品表现** | 新品销售占比 | 新品GMV/总GMV | < 20% |
| | 爆款贡献度 | TOP3产品GMV/总GMV | < 40% |

### 2.2 趋势分析模块

**销售趋势图（近30天）**
- 日销售额折线图
- 7日移动平均线
- 同期对比曲线
- 目标线标注

**渠道占比变化（近90天）**
- 堆叠面积图：天猫/京东/抖音/小红书/线下
- 占比变化标注

**产品生命周期分布**
- 饼图：新品导入期/成长期/成熟期/衰退期
- 各阶段销售额贡献

### 2.3 异常预警模块

```markdown
🔴 红色预警（立即处理）
- 单日销售额 < 目标值70%
- 核心渠道流量下跌 > 30%
- 库存周转率 < 2次/月

🟡 黄色提醒（关注跟踪）
- 周销售额 < 目标值85%
- 新品上市7天销量 < 预期50%
- 客单价连续3天下降
```

## 五、指标体系设计

### 5.1 指标分层

#### 5.1.1 结果指标（Lagging Indicators）
- 销售额、利润、市场份额
- 反映过去业绩，用于评估

#### 5.1.2 过程指标（Leading Indicators）
- 流量、转化率、客单价
- 预测未来业绩，用于优化

### 5.2 核心指标字典

| 指标名称 | 定义 | 计算公式 | 数据来源 | 更新频率 | 负责人 |
|----------|------|----------|----------|----------|--------|
| GMV | 成交总额 | Σ(订单金额) | 各渠道后台 | 实时 | 数据分析师 |
| 转化率 | 访客支付比例 | 支付人数/访客数 | 渠道后台 | 每日 | 运营 |
| 客单价 | 平均订单金额 | GMV/订单数 | 渠道后台 | 每日 | 运营 |
| 复购率 | 老客回购比例 | 复购人数/总客数 | CRM系统 | 每周 | 客服 |
| 库存周转率 | 库存流动速度 | 销售成本/平均库存 | ERP系统 | 每月 | 供应链 |

### 5.3 指标预警规则

```yaml
预警等级:
  红色预警:
    - 连续3天未达标
    - 同比下降>20%
    - 核心指标跌破历史最低
  黄色提醒:
    - 单日未达标
    - 环比下降>10%
    - 趋势持续恶化
  绿色正常:
    - 达成率>95%
    - 同比/环比增长
    - 趋势稳定或向好
```

## 八、报告使用指南

### 8.1 每日使用流程

**早上9:00 - 查看昨天的日报**
1. 打开日销售监控表
2. 检查KPI达成情况
3. 查看异常预警
4. 定位问题渠道/产品

**上午10:00 - 问题追踪**
1. 打开渠道分析报表
2. 分析异常原因（流量?转化?客单价?）
3. 联系对应渠道负责人

**下午15:00 - 调整优化**
1. 根据数据调整推广策略
2. 更新素材/活动
3. 记录调整动作

### 8.2 每周使用流程

**周一早上 - 周会准备**

**周五下午 - 复盘总结**
