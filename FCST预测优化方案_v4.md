# FCST 预测优化方案 v4.0

## 一、核心方法

### Step 1：自下而上（逐SKU趋势）
```python
每个SKU的预测 = 历史加权均值 × 季节系数 × (1 + 该SKU独自的趋势)
```
- 热款：该SKU趋势为正 → 预测高
- 滞销款：该SKU趋势为负 → 预测低
- 新品/停产：按特殊规则处理（不变）

### Step 2：总量校准
```python
合计 = ∑ 各SKU自下而上预测
总量目标 = 上月销售额 × 整体增长率系数
差距 = 总量目标 − 合计
→ 将差距按各SKU的比例分摊回去
```
确保整体达成率 ≥ 95%

### Step 3：准确定义

| 指标 | 定义 | 目标 |
|:---|:---|:---:|
| 准确率 | 单个SKU：\|预测-实际\|/实际 ≤ 20% 算准确 | ≥ 80% |
| 重点款误差 | 重点28个SKU：\|预测-实际\|/实际 ≤ 3% | 重点款全部达标 |
| 系列级误差 | 各系列：合计与实际偏差 ≤ ±5% | 全部达标 |
| 整体达成率 | 总实际/总预测 | ≥ 95% |
